## 1.前言

​	根据`umi.js`官网快速上手流程，利用官方的脚手架，进行项目的搭建。

## 2.搭建项目流程

### （1）环境准备

 node 版本是 10.13 或以上

### （2）脚手架

**此条的前提**：

- 使用 `yarn` 管理 `npm` 依赖作为前提

- 选择好项目所在文件位置

**具体命令**：

```
$ yarn create @umijs/umi-app
# 或 npx @umijs/create-umi-app
```

此操作之后会自动生成文件目录，后续操作便是常规的安装依赖以及启动项目

> 使用脚手架过程中会出现由于网络不佳而retry的情况，只需要检查网络连接，会自动恢复

### （3）安装依赖&启动项目

安装依赖

```
$ yarn
```

启动项目

```
$ yarn start
```

## 3.打包&发布

### （1）打包

```
$ yarn build
```

### （2）本地验证
